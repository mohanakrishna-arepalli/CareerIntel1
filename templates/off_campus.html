Off-Campus Opportunities

Amazon ML Summer School

An initiative by Amazon to provide students with the opportunity to learn key Machine Learning (ML) technologies and skills. Participants will get a chance to attend virtual lectures and solve real-world problems using ML.

JP Morgan Code for Good

JP Morgan's Code for Good hackathon allows participants to collaborate with other coders and use their skills to solve real-world problems for nonprofit organizations. It's a great way to gain experience and make a positive impact.

Learn more

Cisco Ideathon

The Cisco Ideathon is a competition designed to engage students in innovative thinking and problem-solving. Participants work on challenges posed by Cisco and have the chance to win prizes and recognition.

Learn more

Google Summer of Code

Google Summer of Code is a global program focused on bringing more student developers into open source software development. Students work with an open-source organization on a 3-month programming project during their break from school.
